Example site on Node.js using Passport.js, Ajax, Angular, Mongodb

On this site you can create your account, set your profile photo, delete you account if you want, write a message to admin or you can logged in system by 'admin' and 'admin' and view all site users. This site uses the session, you can be in the system for a certain time. At the end of your session, you will need to re-login the system. Pages that use the session are homepage, gallery, video
